POP

移动端弹出层组件的解决方案

View on GitHub Download .zip Download .tar.gz

本弹层插件基于Native Javascript实现，不依赖任何第三方组件。提供了与常用弹层组件一致的语法，同时也提供了简写模式，封装了一些特殊实例，您可以按照需求对pop.open()方法进行个性化量身定制。

通用写法

pop.open({
  content:"谢谢你的支持",
  time:2000,
  style:"color:#fff;background-color:#15b0de"
});

精简写法

写法1：pop.toast("谢谢你的支持");
写法2：pop.toast(["谢谢你的支持"，2000]);
写法3：pop.toast("谢谢你的支持","color:#fff;background-color:#15b0de");

通用写法

pop.open({
  content:"谢谢你的支持",
  time:2000,
  mask:false,
  position:"bottom"
});

精简写法

pop.toast(["谢谢你的支持",3000,false,"bottom"]);

通用写法

pop.open({
  title:["信息","color:#fff;background-color:#383838"],
  content:"这是一个信息框",
  btn:["确定"],
  fun:function(){
    pop.toast("当前版本号:"+pop.v);
  }
});

精简写法

pop.alert(["信息框","这个是替换alert方法是提示框","确定"],function(){
  pop.toast("是不是很惊喜");
});

通用写法

pop.open({
  title:"小青年博客",
  content:"你觉得这个弹层组件怎么样？",
  btn:["很好","一般"],
  fun:[
    function(){
	 pop.toast("谢谢你的支持",1000);
    },
    function(){
	  pop.alert(["提示框","很遗憾没有得到你的肯定","确定"],function(){
		pop.toast("我会再接再厉！");
	  });
    }
  ]
})

精简写法

方法1：（默认动画）
pop.loading();
方法2：（自定义）
pop.loading({
  content:"加载中",
  time:3000,
  style:"background-color:#fff"
})

精简写法

pop.select({
  title:["选择框","color:#fff;background-color:#15b0de"],
  btn:["回复","转发","删除","取消"],
  fun:[
    function(){
    	pop.toast(["回复",2000,false,"bottom"]);
    },
    function(){
        pop.toast(["转发",2000,false,"bottom"]);
    },
    function(){
        pop.toast(["删除",2000,false,"bottom"]);
    },
    function(){
        pop.toast(["取消",2000,false,"bottom"]);
    }
  ]
});
